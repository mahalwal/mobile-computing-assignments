A simple quiz app T/F type of questions. This app lets you save your progress and upload your answers to a local server.

Tools:
   1. For saving and retrieving the responses of a user: SQLite Database
  2. For hosting a local server: apache2 using xampp
  2. For uploading the responses to a local server: PHP and OKHTTP3 API


Note: References -wherever taken- have been cited in the /src files.
